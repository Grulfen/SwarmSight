-Extract the Activity Buildup Example.zip file to a folder
-Open SwarmSight software
-At the top of the screen, browse to the folder where the Activity Buildup Example.mov was extracted
-Click the "Play" button
-Once the video is finished, click "Save as CSV"
-The .CSV file with the frame-by-frame activity levels will be located in the same folder as the video
-Look at the Activity Buildup Analysis.xlsx file for an example of analyzing the activity data
